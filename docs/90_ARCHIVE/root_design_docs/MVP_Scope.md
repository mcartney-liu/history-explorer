# History Explorer - MVP Scope

Version: 1.0

Status: Active


# 1. Purpose

The MVP aims to validate the core idea of History Explorer:

Can users explore history through connected knowledge instead of traditional information searching?


The MVP focuses on proving:

Explore → Connect → Understand → Discover


# 2. MVP Product Goal


Primary Goal:

Create a working historical exploration experience.


Users should be able to:

- Start from a historical topic.
- Explore related entities.
- Understand relationships.
- Continue discovering new paths.


# 3. Target MVP Users


Primary:

History explorers.

Characteristics:

- Curious about history.
- Interested in connections.
- Prefer exploration over memorization.


Secondary:

Students and lifelong learners.


Not the initial focus:

- Professional historians.
- Academic researchers.


# 4. MVP Core Experience


The MVP user flow:


Start with a historical topic

↓

View historical overview

↓

Explore related entities

↓

Discover relationships

↓

Ask AI for explanations

↓

Continue exploration


# 5. MVP Included Features


## 5.1 Historical Topic Exploration

Users can explore:

- Events.
- People.
- Civilizations.
- Locations.


Capabilities:

- Basic historical information.
- Related entities.
- Exploration entry points.


---

## 5.2 Timeline Exploration


Users can understand:

- When events happened.
- What happened before.
- What happened after.
- What happened at the same time elsewhere.


---

## 5.3 Relationship Discovery


Users can discover:

- Event relationships.
- Person relationships.
- Civilization relationships.


Goal:

Make historical connections visible.


---

## 5.4 AI Historical Guide


AI provides:

- Historical explanations.
- Context.
- Relationship interpretation.
- Suggested exploration directions.


AI positioning:

Guide, not authority.


---

## 5.5 Basic Exploration Visualization


MVP visualization:

- Timeline view.
- Basic relationship view.
- Basic geographic context.


# 6. MVP Knowledge Scope


The MVP does NOT require all human history.


Initial knowledge scope:

- Selected historical periods.
- Representative civilizations.
- Important historical events.
- Key historical figures.


Priority:

Depth of exploration > Quantity of information.


# 7. MVP Not Included


The following are NOT MVP scope:


## Social Features

- User community.
- Comments.
- Following.
- Sharing platform.


## Creator Platform

- User-generated historical content.
- Expert publishing.


## Advanced AI Features

- Autonomous historians.
- Fully generated history books.
- Unlimited AI research.


## Advanced Visualization

- VR history.
- 3D world simulation.
- Complex animations.


## Commercial Features

- Subscription system.
- Marketplace.
- Advertising.


# 8. MVP Success Criteria


The MVP succeeds if users can:


1. Start exploration easily.

2. Discover unexpected historical connections.

3. Spend meaningful time exploring.

4. Understand historical context better.

5. Want to continue exploring.


# 9. MVP Metrics


Initial metrics:


Exploration Depth:

Number of meaningful exploration steps.


Exploration Time:

Time spent exploring historical connections.


Return Interest:

Whether users want to continue exploration.


Discovery Rate:

Whether users discover related topics.


# 10. Future Expansion


After MVP validation:


Possible expansion:

- Expert participation.

- Advanced visualization.
