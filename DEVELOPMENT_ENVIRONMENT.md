# History Explorer Development Environment

## Overview

Development environment guide for History Explorer.

## Project Modules

- **frontend/** — React + TypeScript + Vite
- **backend/** — Python + FastAPI
- **ai/** — AI capability foundation
- **data/** — Historical data foundation

## Development Workflow

**Frontend development:**

Run frontend independently.

**Backend development:**

Run backend independently.

**Future:**

Frontend communicates with backend APIs.

## Environment Principles

- Keep modules independent.
- Document configuration changes.
- Avoid unnecessary dependencies.
- Maintain reproducible development.
